# Tax Audit Management System (Frontend)

A web-based frontend system for managing tax audit activities.

## Tech Stack

- React (Vite)
- TailwindCSS v3
- React Router
- JavaScript

## Features

- Dashboard
- Audit Case Management
- Audit Plan Management
- Desk Audit Execution
- Document Upload
- Audit Findings

## Project Structure

src/
components/
pages/
layouts/
routes/

## Installation

Clone the repository

git clone https://github.com/Oliyad-oli/tax-audit-management-frontend.git

Install dependencies

npm install

Run the project

npm run dev
